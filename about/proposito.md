# Proposito do Organon

Organon nasceu para responder uma dor simples e profunda:
quando tudo esta espalhado, a mente trabalha mais para lembrar do que para criar.

A intencao nao e apenas organizar tarefas.
E devolver presenca, clareza e direcao para o dia.

Cada tela existe para reduzir ruido.
Cada acao existe para aproximar voce do que importa.
Cada registro existe para preservar contexto e diminuir recomecos.

Organon acredita que produtividade real nao e correria.
Produtividade real e continuidade com sentido.

Quando o trabalho, os compromissos e a vida pessoal convivem num fluxo unico,
voce ganha mais do que eficiencia.
Voce ganha tranquilidade para decidir melhor.

No fim, o objetivo e humano:
menos caos interno, mais intencao no cotidiano.
